# 🎓 Student Performance Prediction System

## 📌 Project Overview

The Student Performance Prediction System is a Machine Learning-based web application developed using Streamlit. The system analyzes student academic and personal attributes to predict student performance and provide recommendations for improvement.

This project combines Data Analysis, Machine Learning, and Interactive Visualization to help understand the factors affecting student academic performance.

---

## 🚀 Features

### 📊 Dataset Analysis
- Dataset Preview
- Dataset Statistics
- Feature Information
- Correlation Analysis

### 📈 Data Visualizations
- Student Performance Distribution
- Study Time vs Final Grade
- Absences vs Performance
- Correlation Heatmap
- Feature Importance Analysis

### 🤖 Machine Learning Models
- Random Forest Classifier
- Decision Tree Classifier
- Linear Regression

### 🎯 Performance Prediction
Predict student performance using:

- Study Time
- Absences
- Previous Grades
- Family Education
- Internet Access
- Free Time
- Health Status
- Alcohol Consumption
- Social Activities

### 💡 Recommendation System
Provides personalized recommendations based on prediction results.
